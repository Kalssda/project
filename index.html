구글 웹 툴킷(Google Web Toolkit, GWT)은 웹 개발자가 복잡한, 자바스크립트 프론트엔드 애플리케이션을 자바로 만들고 관리할 수 있게 하는 오픈 소스 도구 집합이다. 몇 가지 네이티브 라이브러리를 포함하여, 모든 것이 자바 소스로 이루어져 있다. 마이크로소프트 익스프레이션 웹(Microsoft Expression Web, 코드명: Quartz)은 마이크로소프트사의 HTML 편집기이자 일반적인 웹 디자인 소프트웨어 제품이다. 무료로 제공되는 소프트웨어이며 현재는 개발이 중단된 익스프레션 스튜디오의 구성 요소이기도하다. 웹 2.0(Web 2.0)이란 개방, 참여, 공유의 정신을 바탕으로 사용자가 직접 정보를 생산하여 쌍방향으로 소통하는 웹 기술을 말한다. 웹 1.0이 인터넷을 통해 일방적으로 정보를 보여주었다면, 웹 2.0은 사용자가 직접 콘텐츠를 생산하여 쌍방향으로 소통할 수 있다.
